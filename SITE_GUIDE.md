# 自分自身 HP 更新ガイド

このファイルは、このサイトをあとから更新しやすくするための運用メモです。
画面に見せる内容や更新手順はここにまとめ、コード内のコメントアウトや更新メモは残さない方針です。

## 基本方針

- 公開中の文言やお知らせは、コード内コメントではなくデータとして管理する
- 更新ルールや補足説明は `SITE_GUIDE.md` に集約する
- ブラウザの開発者ツールで見えて困るコメントアウトはコードに残さない

## 主な更新ファイル

### `index.html`

- トップページ本体
- お知らせ一覧の表示枠
- `me` ギャラリーへの導線

### `news-data.js`

- トップページのお知らせデータ
- 配列の上から新しい順に表示される

現在の形式:

```js
window.newsItems = [
  { date: "2026/03/25", text: "自撮りを公開しました。" },
  { date: "2026/03/23", text: "ホームページを公開しました。" }
];
```

ルール:

- 日付は `YYYY/MM/DD`
- 先頭に最新のお知らせを書く
- コメントアウトで旧データを残さない

### `me.html`

- 自撮りギャラリーの公開ページ
- 総自撮り枚数
- ページ送り UI

### `functions/api/me.js`

- `me` 公開API
- 画像一覧とページ情報を返す

### `assets/selfies-public.js`

- `me` ページの一覧描画
- 総枚数表示
- ページ送り描画

### `assets/selfies.css`

- `me` ページの見た目
- 一覧グリッド
- ページ送り

## me ギャラリー運用メモ

- 1ページあたり 100 枚表示
- 上下にページ送りを表示
- 総自撮り枚数を一覧上部に表示
- 公開一覧にはコメント本文は出さない

## お知らせ更新手順

1. `news-data.js` を開く
2. 先頭に新しいお知らせを追加する
3. 古いお知らせを残したい場合はそのまま下に残す
4. 不要になったら行ごと削除する

例:

```js
window.newsItems = [
  { date: "2026/03/25", text: "自撮りを公開しました。" },
  { date: "2026/03/23", text: "ホームページを公開しました。" }
];
```

## 更新時にやらないこと

- HTML コメントで更新メモを書く
- JS のコメントアウトで古いお知らせを残す
- 「あとで消す前提」の文言を本番コードに残す

## 公開前の確認

- トップページのお知らせが正しい順で出る
- トップページの `me` 導線が重複していない
- `/me` で総枚数が出る
- `/me` でページ送りが見える
- `/me?page=2` などで他ページへ移動できる
